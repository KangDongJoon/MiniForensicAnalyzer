# MiniForensicAnalyzer

C# WPF 기반의 간단한 파일 분석 도구입니다.
바이너리 파일을 열어 Hex 데이터 확인, SHA256 해시 계산, 문자열 추출 기능을 제공합니다.

- 파일 열기
- 파일 정보 확인
- Hex Viewer
- SHA256 해시 계산
- ASCII 문자열 추출
